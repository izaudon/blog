## スケール
ドレミファソラシド、という並びは「Cメジャースケール」と呼ばれる規則に沿って並んでいます。


### 音の名前
我々が普段一番耳にする音の名前は、「ドレミファソラシド」だと思いますが、この他にもピアノの鍵盤や音高を表わす名前があります。

  + アルファベットだと「CDEFGABC」
  + 日本語だと「ハニホヘトイロ」
  + ドイツ語だと「CDEFGAH」 など。

ここからは基本的にアルファベットで記述します。


### ピアノの鍵盤
ピアノの鍵盤1オクターブは、CからBまで計12音を経て次のCになる構成になっています。

この中で CDEFGAB はピアノの白鍵部分ですが、黒鍵部分は、例えばCの次の黒鍵は「C# (シャープ)」「Db (フラット)」のように表します。

これを **異名同音** といい、どちらの表し方を使っても良いですが、場面によってどちらを使うかがある程度決まっています。


### スケール
ピアノの鍵盤12音の中で、大半の曲は基本的にこの中の7音を使って曲を作ります。

これを **スケール** といいます。

例えばこの CDEFGAB という白鍵だけを使っている長調のスケールは「Cメジャースケール」といいます。

ピアノの白鍵の並びを見ると、CからDへは全音、DからEは全音、EからFは半音、のように鍵盤によって間隔が違います。

この

  + 「全全半全全全半」という並びを **メジャースケール (長調)** といいます。

また別に、

  + 「全半全全半全全」という並びを **マイナースケール (短調)** といいます。

例えば、同じく白鍵で作られている短調の「Aマイナースケール」の構成音は、「ABCDEFG」です。

この、CメジャースケールならC、AマイナースケールならAが **基準の音 (主音)** です。


### ディグリーネーム
スケールの中の7つの音に、それぞれ1～7の番号をつけます。

スケールの主音を1とします。

例えば、Cメジャースケール (CDEFGAB) ならCを1として、1234567です。

この数字をディグリーネームといい、基本的にローマ数字で表します。


## 音程 ・ 音高
音と音がどれだけ離れているか、ということを度数で表し、「n度離れている」というように言います。

スケール内の音を数字で表したときとほぼ同じです。

Cを基準としたとき、「+半音nコ m度」のように書くと、

  + **C   ->  +0   完全1度 (同じ音)**
  + C#  ->  +1   増1度
  + Db  ->  +1   短2度
  + **D   ->  +2   長2度**
  + D#  ->  +3   増2度
  + Eb  ->  +3   短3度
  + **E   ->  +4   長3度**
  + **F   ->  +5   完全4度**
  + F#  ->  +6   増4度
  + Gb  ->  +6   減5度
  + **G   ->  +7   完全5度**
  + G#  ->  +8   増5度
  + Ab  ->  +8   短6度
  + **A   ->  +9   長6度**
  + A#  ->  +10  増6度
  + Bb  ->  +10  短7度
  + **B   ->  +11  長7度**
  + **C   ->  +12  完全8度 (1オクターブ)**

という風になります。

この後も、1オクターブ上のDは長9度、というように続きます。

特にスケール内では、主音を基準として、Cは1度の音、Eは3度の音、のように簡単に言う場合が多いです。

注意したいのは、カラオケなどでいう「キー+5」は半音5つ分上げることですが、「5度上」と言った場合 **半音+7コ分** 上げることになります。


## ダイアトニックコード
### コード (和音)
コードとは、音の重なりのことです。
特に3つ以上の場合を言うことが多いです。

例えば「C, E, G」ならC(メジャー)のコード、というように音の重なりごとに名前がついています。

通常使われるコードは、1つの音の上に3度ずつ音を重ねてできる和音 (3度堆積和音) です。

例えばCメジャーのコードなら、Cの3度上のE、Eの3度上のGの3つを合わせた和音です。

この音同士の間隔によってコードの名前が決まっています。

  + 1の音、長3度 (+4) の音、完全5度 (+7) の音の並びは **メジャーコード**
  + 1の音、短3度 (+3) の音、完全5度 (+7) の音の並びは **マイナーコード**

のようになります。

全てのコードを説明するのは難しいので各自で調べてください。

### 7thコード ・ テンション
1、3、5のコードに7の音を足したものを **セブンスコード** といいます。

  ![]({{ "/assets/images/2026-06-15/4.png" | relative_url }})

そして7の音以降にも音を足し、コードの機能はそのままに少し雰囲気をオシャレにする音のことを **テンション** といいます。

テンションは主に9thや11thや13thなどで、それぞれ#やbをつけられるものもあります。

また、7th無しで9thを足したものは add9 などのように呼ばれます。


### ダイアトニックコード
この無数にあるコードの中で、スケール内の音だけで作られるコードのことを **ダイアトニックコード** といいます。

例えばCメジャースケールなら、「**C, Dm, Em, F, G, Am, Bm(b5)**」という風になります。
(Bの第3音は完全5度ではなく、5がフラットしています)

これをディグリーネームで表す際、メジャーコードを大文字、マイナーコードを小文字という風に書かせてください。
(クラシックでは両方を区別せず大文字で書きます)

メジャー、マイナーそれぞれのダイアトニックコードは、

  + メジャースケール: **I, ii,     iii, IV, V, vi, vii(b5)**
  + マイナースケール: **I, ii(b5), III, iv, v, VI, VII**

です。
