## 音の名前
我々が普段一番耳にする音の名前は「ドレミファソラシド」だと思いますが、この他にもピアノの鍵盤や音高を表わす名前があります。

  + アルファベットだと「CDEFGABC」
  + 日本語だと「ハニホヘトイロハ」
  + ドイツ語だと「CDEFGAHC」 など。

ここからは基本的にアルファベットで記述します。


## ピアノの鍵盤
ピアノの鍵盤1オクターブは、CからBまで計12音を経て次のCになる構成になっています。

この中で CDEFGAB はピアノの白鍵部分ですが、黒鍵部分は、例えばCの次の黒鍵は「C# (シャープ)」「Db (フラット)」のように表します。

これを **異名同音** といい、どちらの表し方を使っても良いですが、場面によってどちらを使うかがある程度決まっています。


## スケール
ピアノの鍵盤12音の中で、大半の曲は「ドレミファソラシド」のように、7音を使って曲を作ります。

これを **スケール** といいます。

例えばこの CDEFGAB という白鍵だけを使っている長調のスケールは「Cメジャースケール」といいます。

ピアノの白鍵の並びを見ると、CからDへは全音、DからEは全音、EからFは半音、のように鍵盤によって間隔が違います。

この

  + 「全全半全全全半」という並びを **メジャースケール (長調)** といいます。

また別に、

  + 「全半全全半全全」という並びを **マイナースケール (短調)** といいます。

例えば、同じく白鍵で作られている短調の「Aマイナースケール」の構成音は、「ABCDEFG」です。

この、CメジャースケールならC、AマイナースケールならAが **基準の音 (主音)** です。


## ディグリーネーム
スケールの中の7つの音に、それぞれ1～7の番号をつけます。

スケールの主音を1とします。

例えば、Cメジャースケール (CDEFGAB) ならCを1として、1234567です。

この数字を **ディグリーネーム** といい、基本的にはローマ数字で表します。


## 音階の音の名前
スケール内の音に数字をつけましたが、それぞれに別名がついています。

  + **1 -> 主音**
  + 2 -> 上主音
  + 3 -> 中音
  + **4 -> 下属音**
  + **5 -> 属音**
  + 6 -> 下中音
  + **7 -> 導音**

特に重要なものは太字になっています。

上でも触れましたが、スケールの1の音は **主音** と呼ばれ、曲のメロディやコードなど全てにおいての基準になっています。

詳しくは解説しませんが、後で重要になってくるのでこういう名前があるということだけ覚えていてください。


## 音程 ・ 音高
音と音がどれだけ離れているか、ということを度数で表し、「n度離れている」というように言います。

スケール内の音を数字で表したときとほぼ同じです。

Cを基準としたとき、「+半音nコ m度」のように書くと、

  + **C   ->  +0   完全1度 (同じ音)**
  + C#  ->  +1   増1度
  + Db  ->  +1   短2度
  + **D   ->  +2   長2度**
  + D#  ->  +3   増2度
  + Eb  ->  +3   短3度
  + **E   ->  +4   長3度**
  + **F   ->  +5   完全4度**
  + F#  ->  +6   増4度
  + Gb  ->  +6   減5度
  + **G   ->  +7   完全5度**
  + G#  ->  +8   増5度
  + Ab  ->  +8   短6度
  + **A   ->  +9   長6度**
  + A#  ->  +10  増6度
  + Bb  ->  +10  短7度
  + **B   ->  +11  長7度**
  + **C   ->  +12  完全8度 (1オクターブ)**

という風になります。

この後も、1オクターブ上のDは長9度、というように続きます。

特にスケール内では、主音を基準として、Cは1度の音、Eは3度の音、のように簡単に言う場合が多いです。

要は、さっきスケール内の7音に数字をつけたものと同じです。

注意したいのは、カラオケなどでいう「キー+5」は半音+5コ分上げることですが、「5度上」と言った場合 **半音+7コ分** 上げることになります。


## 平行調と同主調
### 平行調
スケールの項で触れましたが、「Cメジャースケール」と「Aマイナースケール」は、同じ白鍵だけで作られたスケールです。

どのメジャースケールにも、同じ音階を共有したマイナースケールが存在します。

この「同じ鍵盤を共有したスケール」を **平行調** (レラティブ) といいます。

平行調は、「CメジャーならAマイナー」「EメジャーならC#マイナー」など、 **短3度** の関係にあります。

### 同主調
「Cメジャースケール」と「Cマイナースケール」のように、主音が同じスケール同士を **同主調** (パラレル) といいます。

メジャースケールを基準に見ると、「3, 6, 7」の音がフラットしています。


## 転調
**転調** とは、曲の途中でスケールが変わることです。

よくラスサビで+1の転調をする曲がありますよね。

+1の転調は「転調したな」というのがハッキリ分かります。

自然な転調をしたい場合は、短3度の「平行調の同主調」へ転調することでシームレスにできます。

#### 例
  + God Knows... / 涼宮ハルヒ (ラスサビ +1)
  + なぜ？謎？！ANSWER / 熊田茜音 ・ 増井優花 (ラスサビ +1)
  + 冬がはじまるよ / 槇原敬之 (サビ +3)
  + きゅうくらりん / いよわ (Bメロ -3)
