## ノンダイアトニックコードを用いたコード進行
ここからは、ノンダイアトニックコードを用いた有名なコード進行を紹介します。

### マリオ進行
  + bVI-bVII-I (マイナースケール)

マリオ進行は、着地する1がメジャーになっている、ピカルディ終止を含む進行です。

スーパーマリオのメインテーマ？地上テーマ？で用いられていることから名付けられた進行です。

メジャーでいうとIV-V-VIで、王道進行の派生とも言えます。

#### 例
  + GO MY WAY!! / 高槻やよい・双海亜美・双海真美
  + もってけ！セーラーふく / 泉こなた・柊かがみ・柊つかさ・高良みゆき

### コンファメーション進行
  + **\|I-vii7(b5)-III7\|vi7-v7-I7\| -> IV**

コンファメーション進行 (コンファメ進行) は、カノン進行の前半をもとにツーファイブを山盛りにした進行です。

ジャズの名曲『Confirmation』で使われていることから名付けられた進行です。

73は6に対するツーファイブ、51は4に対するツーファイブです。

「\|IV-I/III-ii-V\| -> I」と続く場合も多く、これだとループさせることができます。

4へ向かうので、王道進行と繋げることができます。

### 例
  + ハレ晴レユカイ / 涼宮ハルヒ・長門有希・朝比奈みくる
  + ときめきシュガー / 放課後ティータイム
  + カレンダーガール / わか・ふうり・すなお from STAR☆ANIS

### オンリーワン進行
  + **I-IV-V-III -> vi**

オンリーワン進行は、『世界に一つだけの花』で使われたことから名付けられた進行です。

145の後の3がメジャーになっていて、次のviへ進行するセカンダリドミナントになっています。

#### 例
  + ふわふわ時間 / 放課後ティータイム
  + シャイニングスター / 魔王魂
  + Let's get a chance / シャイニーカラーズ

### 丸サ進行 (Just the Two of Us進行)
  + **IV-III-vi-(I)** (メジャースケール)
  + bVI-V-i-(bIII) (マイナースケール)

丸サ進行は、『丸の内サディスティック』で使われたことから名付けられた進行です。

ジャズの名曲『Just the Two of Us』で使われていることから、こちらの名前でもよく見ます。

これは、「安定-不安定-安定-不安定」をの響きを交互に繰り返すのに加えて、viとIによってメジャーともマイナーとも言えないような構成になっているアンニュイな進行です。

viとIの間には、リレイテッド2のvを挟むことが多いです。
(原曲の『Just the Two of Us』もそう)

日本で最も有名な進行の一つなので、情報も使われている曲も死ぬほど出てきます。

### ラインクリシェ
  + vi-bVIaug-I/V-#iv7(b5) -> IV (or ii)

ラインクリシェは、固定のコードの中で、どれか1つの音を半音で動かす進行です。

例で挙げた進行は、CメジャースケールでいうとCとEが固定で、Aの音が半音ずつF#まで下がって、最後にFのコードに向かっています。

下降クリシェだけでなく、上昇クリシェもあります。

#### 例
  + Snow halation / μ's (サビ後半)
  + 新・チョコレート事件 / ゆな from STAR☆ANIS (イントロ、Aメロ1)
  + 恋するみたいなキャラメリゼ / えり・れみ from STAR☆ANIS (サビ 上昇)

### 落ち葉進行
  + \|I-(X)/VII-(X)/bVII-VI\|ii-(X)/#I-(X)/I-V\|

(X)の部分は不定です。

これもベースライン下降・ラインクリシェの一つです。

#### 例
  + 輝く世界の魔法 / CINDERELLA GIRLS
  + Let's go！スマイルプリキュア！ / 池田彩
  + ファーストプロット / 夏川椎菜

### ラムネ進行
  + (X)-III-v-vi

ラムネ進行は、『ラムネ色 青春』に使われたことから名付けられた進行です。

これは、「III -> vi」と「v -> vi」が共存していて、それぞれのコードが同じところに解決している？ という解釈ができるコード進行です。

#### 例
  + めざせポケモンマスター / 松本梨香
  + アイカツメロディ！ / わか from STAR☆ANIS & るか・せな from AIKATSU☆STARS!
