## コード進行
コード進行とは、名の通りコードからコードへ進むことです。


## ダイアトニックコードを用いたコード進行
ここからは具体的なコード進行の例を挙げていきます。

特に記載が無い場合はメジャースケールの話だと思ってください。

### ツーファイブワン (251)
  + **ii7-V7-I**

ツーファイブワンは、名前の通り251の進行です。

2から5の進行は強進行、5から1の進行はドミナントモーションになっています。

ジャズのスタンダードと呼ばれる進行ですが、それ以外のジャンルでも積極的に使われています。

### ポップパンク進行
  + **I-V-vi-IV**
  + vi-IV-I-V
  + IV-I-V-vi

ポップパンク進行は、世界で一番使われているとも言われるコード進行です。

基本形の1564の他にも、順番をずらした派生形もあります。

循環して半永久的に続けることができます。

#### 例
  + Let It Be / ビートルズ (1564)
  + Lemon / 米津玄師 (4156)
  + 前前前世 / RADWINPS (6415)

### 王道進行
  + **IV-V-iii-vi**
  + IV-V-vi-iii
  + IV-V-vi
  + ...

王道進行は、日本で一番使われているコード進行です。

派生形は多すぎるので省略します。

これも循環して半永久的に続けることができます。

また、終わりの6が2へ強進行できることから、王道進行 -> ツーファイブワンへと滑らかに進行することができます。

#### 例
  + もう恋なんてしない / 槇原敬之 (4536 -> 251)
  + アルストロメリア / アルストロメリア (45365)

### 小室進行
  + **vi-IV-V-I**

小室進行は、主に小室哲哉氏がよく使ったコードです。

これも循環して半永久的に続けることができます。

#### 例
  + Get Wild / TM NETWORK
  + only my railgun / fripside
  + オーバーライド / 吉田真世

### カノン進行
  + **\|I-V-vi-iii\|IV-I-IV-V\| -> I**
  + \|I-III-vi-V\|(省略)\|
  + \|(省略)\|IV-I-ii-V\|

カノン進行は、名の通りパッヘルベルのカノンで全編通して使われている進行です。

転回形を用いることで、ベース音を「C-B-A-G\|F-E-D...」と滑らかに下降することができます。

派生に出てくるメジャーのIIIについては次回解説します。

#### 例
  + どんなときも。 / 槇原敬之 (Aメロ)
  + ハートにヒント！名探偵プリキュア！ / 石井あみ (1561\|4125)
  + アイドル活動！ / わか・ふうり・すなお from STAR☆ANIS (13651 -> 4536 -> 251)

### ZUN進行
  + bVI-bVII-i (マイナースケール)

ZUN進行は、東方pojectのほぼ全ての楽曲に使用されているコード進行です。

メジャースケールにおける王道進行です。
