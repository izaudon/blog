## 効果的な転調
以前の回でも話しましたが、転調とは曲の途中で別のスケールに変わることです。

## +1転調
+1転調は、手軽に使えるのに加えて共有する鍵盤がかなり少ないので、「雰囲気が変わったな」というのが分かりやすいです。

半音転調が起きるタイミングでは、「転調前のV -> 転調後のV」の進行が定番です。

### 例 人類みなセンパイ！
  + 人類みなセンパイ！ / アズマリム

ラスサビでDキーからEbキーへの+1転調、俗にいう言い直し転調というやつです。

転調の際、「A7sus4-A7 -> Bb7sus4-Bb7」の進行があります。


## +2転調
+2転調は、+1に比べて共有する鍵盤が多いので、そこまで露骨な転調になりづらいです。

全音転調の定番は、「VIsus4-VI」の後に転調です。

これは、転調後から見ると「Vsus4-V」に見え、Iへ進むことで強進行が起き、「いつの間にか転調している」感覚があります。

このような「転調前と後のどちらからも解釈できるコード」を **ピボットコード** といいます。

### 例1 カウントダウンラブ
  + カウントダウンラブ / I'm a Cutie Finder

ラスサビでGキーからAキーへの+2転調があります。

ここで、「Esus4-E -> A」の進行がありますが、EのコードはGキーから見ると6、Aキーから見ると5になります。

### 例2 ラビューダ♡トライアングル
  + ラビューダ♡トライアングル / 島村卯月・宮本フレデリカ・喜多日菜子

Bメロからサビへ、DキーからEキーへの+2転調で、「Asus4-B -> E」の進行があります。

これは例1とは違い、普通のVのsus進行「Asus4-A」になるところを、途中でVIのBへ進むことで転調することを教えてくれています。

やはりBは、Dキーから見て6、Eキーから見て5にあたります。


## -3転調
-3転調は、平行調の同主調への転調です。

かなり近い関係にある調なので、非常に自然な転調ができます。

定番は「メロ終わりのVからサビ頭のVI」へ進行し、これが転調後のIになります。

### 例1 Let's アイカツ！
  + Let's アイカツ！ / るか・もな・みき from AIKATSU☆STARS!

Cメロから落ちサビへの転調で、EキーからDbキーへの-3転調があります。

「メロ終わりのAb (EキーのV) -> サビ頭のDb (EキーのVI/DbキーのI)」になっています。

### 例2 カレンダーガール
  + カレンダーガール / わか・ふうり・すなお from STAR☆ANIS

Cメロから落ちサビへの転調で、GbキーからEbキーへの-3転調があります。

これも「メロ終わりのDb (GbキーのV) -> サビ頭のEb (GbキーのVI/EbキーのI)」になっています。

この曲に関しては、「Bb (V of Eb)」が間に挟まっているので、+1転調のときと同じ「前のV -> 後のV」も起きています。


## +3転調
+3転調は、同主調の平行調への転調です。

-3と同じように、非常に自然な転調ができます。

定番は同主短調から借りてきた「iv-bVII」が転調後のIへ進行します。

### 例1 Let's アイカツ！
  + Let's アイカツ！ / るか・もな・みき from AIKATSU☆STARS!

Aメロの途中で、EキーからGキーへの転調です。

ここで起きている「Am-D -> G」は、転調後から見るとツーファイブワンになっています。

転調前から見ると「iv-bVII -> bIII」です。

### 例2 冬がはじまるよ
  + 冬がはじまるよ / 槇原敬之

Bメロからサビへ向かう際に+3転調が起きていますが、これはAキーからAmキーへの転調です。

「Dm-G -> Am」は、転調後から見るとツーファイブワンで、転調前から見ると「iv-bVII -> i」です。


## その他の転調
その他の転調でも基本的にはセカンダリドミナントがキッカケに用いられていることが多いと思います。

### 例 Trancing Pulse
  + Trancing Pulse / 渋谷凛・神谷奈緒・北条加蓮

イントロ (サビ) からAメロへ+6、Bメロからサビへ+6転調が起きています。

+6 (増4度/減5度) は、五度圈でいうと対角に位置し、共有する鍵盤が+1と同様かなり少ないです。

+6のメリットは、世界観がガラっと変わるのに加えて、+6と-6は同じなので、同じ方法の転調で戻れるというのもあるかもしれません。

この曲は、おそらくCmキーとAキーの関係が「Cm -3(Am) -3(A) / A +3(Am) +3(Cm)」で「同主調の平行調の同主調」になっています。

マジで初心者ですみませんが、サビがメジャーなのかマイナーなのか分かりませんでした。

たぶん曲終わりの「Asus4 -> A」がIのメジャーなんじゃないかな～と思っていますが……。

イントロからAメロへ、AキーからCmキーの転調が起きる際「Gsus4-G -> Cm」が起きています。

その前も見ると、+3転調のときと同様「iv-bVII」の流れが出てきており、それが今回はマイナーの「biii」に進行しています。

転調後のCmキーから見ると「V -> i」になっています。

Bメロからサビへは逆にCmキーからAキーです。

ここで起きている「Asus4-A -> D」は、転調後のAキーから見ると「I -> IV」です。
